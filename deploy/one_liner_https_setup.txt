# One-liner HTTPS Setup Command
# Copy and paste this entire command into your GCP VM terminal:

curl -fsSL https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/main-project/main/deploy/run_https_setup_from_github.sh | bash -s -- YOUR_GITHUB_USERNAME

# Replace YOUR_GITHUB_USERNAME with your actual GitHub username
# Example: curl -fsSL https://raw.githubusercontent.com/hinah/main-project/main/deploy/run_https_setup_from_github.sh | bash -s -- hinah
